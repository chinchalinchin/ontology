# Ontology

*Ontology* is a personal project I have been working on in some form over the past decade. It has been scrapped and remade many times as I learned the mechanics of game development and software engineering. This is my latest attempt to build a highly configurable game engine. The general philosophy of the project is: simulation and emergence. The goal of the engine is to provide a process that ingests a starting world state and then evolves according to its own internal logic, without scripting. All gameplay is achieved by configuring the initial conditions of the world state.

!!! note "LPC"
    All assets are taken from the [Liberated Pixel Cup](https://lpc.opengameart.org/)
